# 二.JVM
## 1.三种垃圾回收算法
- 标记清除算法：快，内存碎片。
- 标记整理算法：慢，无内存碎片，老年代。
- 复制算法：快，但内存利用不全，新生代。
## 2.四种垃圾回收器
- 串行垃圾回收器：单线程，回收时阻塞其他线程。
	- 新生代复制，老年代标记整理。
- 并行垃圾回收器：多线程，多个线程一起回收。
	- 新生代复制，老年代标记整理。
- CMS并发垃圾回收器：针对老年代，初始标记阻塞，然后一个线程进行并发标记，然后多个线程重新标记，最后一个线程并发清理，速度快。
- G1垃圾回收器：划分多个Region（edon，survivor，old，humongous），采用复制算法。
