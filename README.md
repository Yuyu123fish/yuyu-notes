# yuyu-notes

这是我的个人技术学习笔记仓库，主要用于记录 Java 后端、计算机基础、算法、前端、AI Agent、项目复盘和面试准备等内容。

笔记主要使用 Obsidian 编写，采用 Markdown 格式进行长期沉淀。  
当前内容仍在持续整理中，部分笔记可能还比较粗糙，会随着学习和项目实践不断补充、修正和重构。

## 内容结构

|   ├── Code              # 语言

├── 08-Cards              # 知识卡片（快速记忆）
