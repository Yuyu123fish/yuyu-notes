public class Main {
    static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode(int val) {
            this.val = val;
        }
    }

    static class Solution {
        int cur = 0;
        int result = 0;

        public int kthSmallest(TreeNode root, int k) {
            dfs(root, k);
            return result;
        }

        private boolean dfs(TreeNode root, int k) {
            if (root == null) {
                return false;
            }

            if (dfs(root.left, k)) {
                return true;
            }

            cur++;
            if (cur == k) {
                result = root.val;
                return true;
            }

            return dfs(root.right, k);
        }
    }

    public static void main(String[] args) {
        TreeNode root = new TreeNode(3);
        root.left = new TreeNode(1);
        root.right = new TreeNode(4);
        root.left.right = new TreeNode(2);

        int k = 1;

        Solution solution = new Solution();
        int ans = solution.kthSmallest(root, k);

        System.out.println(ans);
    }
}

## lc128.最长连续序列
时间复杂度要`O(n)`，所以使用`Set`，遍历一次数组，然后在`set`的基础上进行判断，由于`set`里面存储的元素并不是连续的，所以必须对每个元素进行判断，发现`num-1`存在就跳过，否则开始进行统计。
```java
class Solution {
    public int longestConsecutive(int[] nums) {
        if(nums.length == 0){
            return 0;
        }
        Set<Integer> set = new HashSet<>();
        for(int num : nums){
            set.add(num);
        }
        int max = 1;
        int cur = 1;
        for(int num : set){
            if(set.contains(num-1)){
                continue;
            }else{
                while(set.contains(num+1)){
                    num++;
                    cur++;
                }
                max = Math.max(max, cur);
                cur = 1;
            }
        }
        return max;
    }
}
```

## lc41.缺失的第一个正数
把每个在 `[1, n]` 范围内的数字放到它应该在的位置 `num - 1`，如果目标位置已经有相同数字，就说明是重复数，不再交换，避免死循环。如果说是重复的数字，对于非连续情况，它会**停在中间**，对于连续情况，它会被**抛到末尾**。
```java
class Solution {
    public int firstMissingPositive(int[] nums) {
        //正数num的真正位置是num-1，由nums[nums[i]-1] != nums[i] != nums[i]判断
        //nums[nums[i]-1]是num原本该在的位置上，现在实际的数
        //nums[i]当前位置的数
        //只有两者相同时，才能判断当前数已经在它合适的位置
        //而且也可以判断两个相同的数出现影分身的情况
        for(int i = 0; i < nums.length; i++){
            while(nums[i] > 0 && nums[i] <= nums.length && nums[nums[i]-1] != nums[i]){
                int index = nums[i]-1;
                int temp = nums[i];
                nums[i] = nums[index];
                nums[index] = temp;
            }
        }
        for(int i = 0; i < nums.length; i++){
            if(nums[i] != i+1){
                return i+1;
            }
        }
        return nums.length+1;
    }
}
```

# 二叉树
## lc145.二叉树后序遍历-迭代
对于无法向左走的情况，就需要进行判断，判断当前是否可以继续向右走
- 无法向右走的情况：
	- `poped.right == null`，右边为空没办法走
	- `poped.right == pre`，右边已经走过了
	- 无法向右走后：
		- `result.add(poped.val)`，添加答案
		- `pre = poped`，更新之前走过的
- 剩下的情况可以继续向右走
	- `cur = poped`，更新cur准备移动
	- `stack.push(cur)`，加入到栈中准备判断
	- `cur = cur.right`，此时`cur`可能可以向左走
```java
class Solution {
    public List<Integer> postorderTraversal(TreeNode root) {
        List<Integer> result = new ArrayList<>();
        Deque<TreeNode> stack = new ArrayDeque<>();
        TreeNode cur = root;
        TreeNode pre = null;
        while(!stack.isEmpty() || cur != null){
            if(cur != null){
                stack.push(cur);
                cur = cur.left;
            }else{
                //准备向右走
                TreeNode poped = stack.pop();
                if(poped.right == null || poped.right == pre){
                    //右边没有或者已经走过了
                    result.add(poped.val);
                    pre = poped;
                }else{
                    //可以向右走
                    cur = poped;
                    stack.push(cur);
                    cur = cur.right;
                }
            }
        }
        return result;
    }
}
```

## lc437.路径总和3
有**三个注意的点**：
1. 遇到前缀和最好用`long`
2. 这道题是单边路径，所以必须在处理完两边后进行**回溯**！
3. 一开始必须`map.put(0L, 1)`，不然前缀和为0的时候无法正常添加
```java
class Solution {
    int result = 0;
    public int pathSum(TreeNode root, int targetSum) {
        Map<Long, Integer> map = new HashMap<>();
        map.put(0L, 1);
        dfs(root, targetSum, 0, map);
        return result;
    }
    private void dfs(TreeNode root, int targetSum, long curSum, Map<Long, Integer> map){
        if(root == null){
            return;
        }
        curSum += root.val;
        long needSum = curSum - targetSum;
        if(map.containsKey(needSum)){
            result += map.get(needSum);
        }
        map.put(curSum, map.getOrDefault(curSum, 0) + 1);
        dfs(root.left, targetSum, curSum, map);
        dfs(root.right, targetSum, curSum, map);
        //回溯
        map.put(curSum, map.get(curSum) - 1);
    }
}
```

## lc236.二叉树的最近公共祖先
核心思路如下：
- `root`节点如果为空，或者为p和q的任何一个就返回，否则到左右子树里面去找
- `left == null`左子树为空，说明在右子树。`right == null`右子树为空，说明在左子树。
- 否则两个都不为空，则`root`就是最近公共祖先，返回`root`。
```java
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if(root == null || root ==p || root == q){
            return root;
        }
        TreeNode left = lowestCommonAncestor(root.left, p, q);
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        if(left == null){
            return right;
        }
        if(right == null){
            return left;
        }
        return root;
    }
}
```

## lc207.课程表
最需要注意的是有些课程是没有后置课程的，如果直接`List<Integer> afterCourses = map.get(curCourse)`得到的很有可能是`null`，所以判空发现为`null`后就跳过。
还有一个点是可以完成的课程`finish`建议在发现可以被完成时统计，而不是在每一轮`while`里面统计。
```java
class Solution {
    public boolean canFinish(int numCourses, int[][] prerequisites) {
        //需要统计 每门课的入度，每门课后置课程
        int[] enterCount = new int[numCourses];
        Map<Integer, List<Integer>> map = new HashMap<>();
        int finish = 0;
        for(int[] prerequisity : prerequisites){
            int a = prerequisity[0];
            int b = prerequisity[1];
            enterCount[a]++;
            if(!map.containsKey(b)){
                List<Integer> list = new ArrayList<>();
                map.put(b, list);
            }
            map.get(b).add(a);
        }
        Deque<Integer> deque = new ArrayDeque<>();
        for(int i = 0; i < numCourses; i++){
            if(enterCount[i] == 0){
                deque.offer(i);
                finish++;//完成了一门课程
            }
        }
        //现在已经找到入度为0的课
        while(!deque.isEmpty()){
            int curCourse = deque.poll();
            if(!map.containsKey(curCourse)){
                continue;
            }
            List<Integer> afterCourses = map.get(curCourse);
            for(int afterCourse : afterCourses){
                enterCount[afterCourse]--;
                if(enterCount[afterCourse] == 0){
                    deque.offer(afterCourse);
                    finish++;//完成了一门课程
                }
            }
        }
        return finish == numCourses;
    }
}
```

            return 0;

## lc139.单词拆分

- 然后从前往后遍历，对于每个`i`我们用`j`从0开始遍历，先检查`dp[j]`能不能被拼出来，再检查索引`j`到`i`的单词是否存在于`HashSet`中即可。

class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> set = new HashSet<>(wordDict);
        boolean[] dp = new boolean[s.length()+1];
        dp[0] = true;
        for(int i = 1; i <= s.length(); i++){
            for(int j = 0; j < i; j++){
                if(dp[j] && set.contains(s.substring(j, i))){
                    dp[i] = true;
                }
            }
        }
        return dp[s.length()];
    }
}

## lc416.分割等和子集
- 每个数只能用一次，是01背包问题，那么我们只能去遍历每一个物品（也就是数）
- 注意，每次遍历都先copy一个dp，然后修改这个临时的dp`tempdp`，对于判断，必须使用原来的dp进行判断，如果使用新的dp进行判断，会导致一个数被反复使用从而变成完全背包问题。
```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for(int num : nums){
            sum += num;
        }
        if(sum % 2 == 1){
            return false;
        }
        int targetSum = sum / 2;
        Arrays.sort(nums);
        boolean[] dp = new boolean[targetSum+1];
        dp[0] = true;
        for(int num : nums){
            boolean[] tempDp = new boolean[targetSum+1];
            System.arraycopy(dp, 0, tempDp, 0, targetSum+1);
            if(num > targetSum){
                return false;
            }
            for(int i = 0; i + num <= targetSum; i++){
                if(dp[i]){//注意只能以原来的dp来判断，因为是01背包
                    tempDp[i+num] = true;
                } 
            }
            dp = tempDp;
        }
        return dp[targetSum];
    }
}
```
